## Running the application
The API requires java 1.8

### On Mac/Linux

```
./gradlew bootRun
```

### On Windows

```
gradlew.bat bootRun
```

The API will run on **port 8080**.

The API is running and queryable when your receive the following output
```
<=========----> 75% EXECUTING [1m 43s]
> :bootRun
```
## Testing the application

### Starting the test runner
